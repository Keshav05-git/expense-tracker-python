# Personal Expense Tracker (CLI) 💸

A lightweight, command-line financial utility application built entirely with core Python. This project demonstrates foundational backend logic, data structuring, and state management by allowing users to dynamically input, categorize, and aggregate daily financial transactions.

# Features

* **Dynamic Data Entry:** Users can log expenses with specific attributes including Date, Category, Description, and floating-point Amounts.
* **Session Data Management:** Utilizes Python lists and dictionaries to store and map user input accurately during runtime.
* **Automated Aggregation:** Features a real-time calculation engine that iterates through the data structure to output total session expenditures.
* **Interactive CLI:** Implements an infinite `while` loop with robust conditional routing to maintain application state until the user explicitly exits.

# Technologies & Concepts Used

* **Language:** Python 3.x
* **Core Concepts:** * Data Structures (Lists, Dictionaries)
    * Control Flow (Loops, If/Elif logic)
    * Input/Output (I/O) handling and Type Casting
    * String Formatting (f-strings)
